## Project Description
This repository contains a modular Research Paper Summarizer system designed to generate structured summaries of academic papers with both expert and lay audience support.

## Repository Structure

- `system_prompt.md` - Main system prompt for the summarizer
- `modules/` - Modular architecture components
  - `01_intake_setup.md` - Input validation and setup
  - `02_section_loop.md` - Section-by-section processing
  - `03_guardrails.md` - Safety and quality checks
  - `04_rendering_refinement.md` - Output formatting
  - `05_citation_extractor.md` - Citation extraction
  - `06_key_contributions.md` - Key contributions identification

## Features
- 150-word summaries per section
- Expert and lay audience summaries
- Citation extraction and validation
- Hallucination prevention
- Missing section detection
